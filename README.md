# Учебный Трекер - Мобильное приложение для управления учебными задачами студентов

🎯 **О проекте**

**Почему был создан этот продукт?**
Учебный Трекер - это практическое мобильное приложение для Android, созданное для эффективного управления учебными задачами и организации учебного процесса. Проект разработан как учебный пример полного цикла разработки Android-приложения с использованием современных технологий и лучших практик.

**Какую проблему он решает?**
Приложение помогает студентам систематизировать учебные задачи, отслеживать дедлайны, анализировать успеваемость и эффективно распределять учебную нагрузку. Больше не нужно держать в голове все задания или использовать разрозненные инструменты - вся учебная информация организована в одном месте в вашем смартфоне.

**В чём его ключевые преимущества?**
- 📱 **Полностью оффлайн**: Все данные хранятся локально на устройстве
- 📊 **Автоматические расчеты**: Процент выполнения, статистика успеваемости
- 🎓 **Умная организация**: Классификация задач по предметам и приоритетам
- 💾 **Надёжное хранение**: Локальная база данных SQLite
- 🎨 **Современный интерфейс**: Material Design, интуитивная навигация

## Технологии

**Минимальные требования:**
- Android 8.0 (API level 26) или выше
- 50 МБ свободного места

**Стек технологий:**
- **Язык программирования**: Java
- **Архитектура**: MVC с элементами MVVM
- **UI Framework**: Android Views + Fragments
- **Локальное хранилище**: SQLite с SQLiteOpenHelper
- **Навигация**: BottomNavigationView + ViewPager2
- **Визуализация**: MPAndroidChart для графиков
- **Компоненты**: RecyclerView, CardView, Material Design Components

## Быстрая установка и запуск

### Установка через APK
1. **Скачайте APK-файл**: [StudentTaskTracker.apk](#)
2. **Настройте Android устройство**:
   - Перейдите в Настройки → Безопасность
   - Включите "Неизвестные источники"
3. **Установите приложение**:
   - Откройте скачанный APK-файл
   - Нажмите "Установить"
4. **Готово!** Приложение появится в списке ваших приложений

## Возможности

### 📝 **Управление учебными задачами**
- Добавление новой задачи с детальной информацией:
  - Название задачи
  - Предмет (выбор из списка)
  - Приоритет (Высокий/Средний/Низкий)
  - Срок выполнения
  - Описание
- Автоматическое обновление статусов

### 📊 **Статистика и аналитика**
- **Процент выполнения** задач
- **Распределение по приоритетам** (круговая диаграмма)
- **Распределение по предметам** (круговая диаграмма)
- **Количество просроченных задач**
- **Ближайшие задачи** (на сегодня/завтра)

### 🔄 **Организация и поиск**
- **Фильтрация задач** по предмету, приоритету, статусу
- **Поиск задач** по названию и описанию
- **Сортировка** по дате, приоритету, статусу
- **Валидация данных**: Проверка обязательных полей

### 🗓️ **Недельное планирование**
- Просмотр задач на текущую неделю
- Навигация по неделям
- Планирование нагрузки на 7 дней

### 📤 **Экспорт данных**
- Экспорт всех задач в CSV-формат
- Сохранение в папку Downloads
- Структурированный формат для анализа

## Использование

### 🏠 **Главный экран (Обзор)**
После запуска приложения вы попадаете на главный экран, где:
- Отображается список всех учебных задач
- Показываются фильтры и поиск
- Доступны кнопки добавления и экспорта задач

### ➕ **Добавление новой задачи**
1. Нажмите кнопку "Добавить задачу"
2. Заполните поля:
   - **Название**: Краткое описание задания
   - **Предмет**: Выберите предмет из списка
   - **Приоритет**: Выберите уровень важности
   - **Срок выполнения**: Установите дату дедлайна
   - **Описание**: Детальное описание (необязательно)
3. Нажмите "Сохранить"

### 📈 **Просмотр статистики**
1. Перейдите на вкладку "Статистика" через нижнее меню
2. На экране статистики отображаются:
   - Общее количество задач
   - Процент выполненных задач
   - Диаграмма распределения по приоритетам
   - Диаграмма распределения по предметам
   - Просроченные и ближайшие задачи

### 🗓️ **Недельное планирование**
1. Перейдите на вкладку "Неделя"
2. Используйте кнопки навигации для перемещения по неделям
3. Просматривайте задачи, запланированные на каждый день недели
4. Отмечайте выполнение задач прямо в недельном виде

### 📤 **Экспорт задач**
1. На главном экране нажмите "Экспорт в CSV"
2. Дождитесь завершения процесса
3. CSV-файл будет сохранён в папке Downloads

### 🔍 **Поиск и фильтрация**
1. Используйте поле поиска для быстрого нахождения задач
2. Применяйте фильтры по:
   - Предмету
   - Приоритету
   - Статусу выполнения
3. Комбинируйте фильтры для точной выборки

### Ключевые компоненты:
1. **DatabaseHelper** - SQLiteOpenHelper для работы с локальной БД
2. **Task** - Модель данных для учебной задачи
3. **OverviewFragment** - Главный экран со списком задач
4. **StatisticsFragment** - Экран статистики с диаграммами
5. **WeekViewFragment** - Недельный просмотр задач
6. **AddTaskDialog** - Диалог добавления/редактирования задач
7. **TaskAdapter** - Адаптер для RecyclerView

## Разработка и тестирование

Проект был разработан с использованием **Android Views и Fragments** для создания пользовательского интерфейса и паттерна **MVC с элементами MVVM** для организации кода. Доступ к данным реализован через локальную базу данных **SQLite** с использованием SQLiteOpenHelper.

### Тестирование
В проекте проведено комплексное тестирование, включающее:

**Модульное тестирование**: Проверка корректности работы отдельных компонентов, таких как расчёт статистики, валидация введенных данных, работа с базой данных, адаптеры и фрагменты.

**Интеграционное тестирование**: Проверка взаимодействия между модулями (например, добавление задачи → обновление статистики → отображение на главном экране). Тестирование проводилось на основе основных пользовательских сценариев, таких как:
- **Т1**: Добавление первой задачи → Расчет статистики → Отображение на главном экране
- **Т2**: Добавление нескольких задач → Расчет процента выполнения → Отображение в статистике
- **Т3**: Фильтрация задач → Экспорт данных → Проверка корректности CSV
- **Т4**: Недельное планирование → Навигация по неделям → Обновление статусов
