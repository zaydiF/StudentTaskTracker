# StudentTaskTracker

## О проекте

StudentTaskTracker появился как практическая реализация цифровой системы управления учебными задачами в формате мобильного приложения для Android. Проект объединяет потребности студентов в структурированном учёте заданий, контроле дедлайнов и анализе успеваемости с интуитивным интерфейсом и локальным хранением данных. Разработка велась как комплексный пример полноценного мобильного приложения с архитектурой на основе фрагментов, реляционной базой данных SQLite и адаптивным пользовательским интерфейсом.

Приложение предоставляет единую цифровую среду для организации учебного процесса без зависимости от облачных сервисов. Оно автоматизирует рутинные операции: создание и редактирование задач, фильтрацию по предметам и приоритетам, отслеживание просроченных дедлайнов, недельное планирование и экспорт данных, заменяя разрозненные заметки и бумажные ежедневники структурированной системой.

### В чём его ключевые преимущества?

- **Полная локальность и безопасность:** Все данные хранятся в локальной базе SQLite на устройстве пользователя
- **Адаптивный интерфейс:** Поддержка различных размеров экранов (телефоны и планшеты)
- **Комплексное управление задачами:** Создание, редактирование, фильтрация, поиск и статистика в одном приложении
- **Автоматизация учебных процессов:** Автоматические уведомления о просроченных задачах, цветовая индикация приоритетов, массовые операции
- **Визуализация успеваемости:** Круговые диаграммы по приоритетам и предметам, прогресс-бар выполнения
- **Готовое рабочее решение:** Полнофункциональное приложение с типовыми сценариями учёта задач «из коробки»

## Технологии

Для работы приложения требуется:

- **Android Runtime Environment:** Android 7.0+ (API 24+)
- **Java Development Kit (JDK):** 8 или выше

### Архитектура проекта:

| Компонент | Технология |
|-----------|-----------|
| Язык и интерфейс | Java, XML Layouts, Material Design 3 |
| База данных | SQLite с полной схемой задач, предметов и настроек |
| Архитектура | Фрагменты (Fragment) с BottomNavigationView |
| Дополнительные компоненты | RecyclerView с адаптером, MPAndroidChart для диаграмм, стандартные Android-компоненты |

## Возможности

### Система задач
- **Создание задач:** Добавление с указанием названия, предмета, приоритета (Высокий/Средний/Низкий), срока выполнения и описания
- **Редактирование:** Изменение всех полей существующей задачи через диалоговое окно
- **Удаление:** Удаление задачи с подтверждением через AlertDialog
- **Отметка выполнения:** Переключение статуса через CheckBox с визуальным зачёркиванием

### Фильтрация и поиск
- **Фильтр по предмету:** Отбор задач по конкретной дисциплине
- **Фильтр по приоритету:** Высокий, Средний, Низкий
- **Фильтр по статусу:** Активные, Выполненные, Все
- **Поиск:** Поиск по названию, предмету или описанию
- **Индикация фильтров:** Цветовая подсветка количества активных фильтров (зелёный/жёлтый/красный)

### Статистика и планирование
- **Общая статистика:** Всего задач, выполненных, активных, процент выполнения
- **Диаграммы приоритетов:** Круговая диаграмма распределения по уровням важности
- **Диаграммы предметов:** Круговая диаграмма распределения задач по дисциплинам
- **Просроченные задачи:** Автоматический подсчёт задач с истёкшим дедлайном
- **Ближайшие задачи:** Задачи на сегодня и завтра с цветовой индикацией

### Недельное планирование
- **Навигация по неделям:** Переключение между неделями кнопками «←» и «→»
- **Фильтрация по неделе:** Отображение только задач текущей недели
- **Статистика недели:** Общее количество, выполненные, активные задачи за неделю

### Массовые операции
- **Отметить все выполненными:** Массовое изменение статуса всех задач
- **Удалить выполненные:** Массовое удаление всех выполненных задач

### Экспорт и уведомления
- **Экспорт в CSV:** Сохранение списка задач в файл CSV в папку Downloads
- **Уведомления о просроченных:** Системное оповещение при запуске приложения
- **Уведомления о сегодняшних:** Напоминание о задачах на текущий день

### Справочник предметов
- **Предустановленный список:** Более 200 дисциплин (математика, физика, программирование и др.)
- **Добавление новых:** Возможность создать собственный предмет
- **Поиск по предметам:** Быстрый поиск при выборе дисциплины

## Использование

### Начало работы
При первом запуске приложение создаёт локальную базу данных и открывает главный экран с пустым списком задач.

### Главный экран
После запуска открывается экран OverviewFragment с:
- Карточками статистики (всего, выполнено, активно)
- Полем поиска и кнопкой очистки
- Спиннерами фильтрации (предмет, приоритет, статус)
- Список задач в RecyclerView с возможностью скролла
- Кнопками быстрых действий (добавить, отметить все, удалить выполненные, экспорт)

### Нижняя навигация

| Раздел | Описание |
|--------|----------|
|  Главная | Список задач, фильтры, поиск, статистика |
|  Неделя | Задачи текущей недели с навигацией |
|  Статистика | Диаграммы, просроченные, ближайшие задачи |

### Работа с задачами
- **Добавление:** Нажать «+», заполнить поля, выбрать предмет и дату, сохранить
- **Редактирование:** Нажать кнопку «редактировать» на карточке задачи
- **Удаление:** Нажать кнопку «удалить» на карточке задачи, подтвердить
- **Выполнение:** Отметить CheckBox на карточке задачи
- **Просмотр деталей:** Нажать на карточку задачи

### Фильтрация и поиск
- Выбрать предмет, приоритет или статус в спиннерах
- Ввести текст в поле поиска
- Нажать «Сбросить фильтры» для очистки всех условий
- Цвет индикатора фильтров показывает количество активных фильтров

### Все изменения сохраняются автоматически через транзакции БД
